<div align="center">
  <h2>Features</h2>
</div>

### MVP

- Browse for cards throughout the history of Magic
- Build decks
- Rate and comment on the creations of other users
---

### Bonus Features
- Statistical analysis of your deck composition
- Hand drawing algorithm
- Track your collection
- Dark Mode
